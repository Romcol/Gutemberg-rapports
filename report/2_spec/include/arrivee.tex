\label{sec:arrivee_recherche}
L’utilisateur ici a accès à un moteur de recherche similaire à celui du site web Amazon qui propose d’effectuer une recherche sur les articles, les journaux ou les revues de presse. L’utilisateur après avoir saisi dans le champ recherche(confère figure ci-dessous) peut clicker sur un des boutons ‘articles’, ‘journaux’ ou ‘revue de presse’ en fonction de ce qu’il veut rechercher. Les résultats de la recherche pourront par la suite être affinés selon plusieurs critères.
